# Installation

Welcome to NexiumUI! Let's get you started with installing and setting up the library.

## Quick Start

```bash
npm install nexiumui
# or
yarn add nexiumui
# or
pnpm add nexiumui
```

## Setup

Import the styles in your root layout:

```tsx
import '@nexiumui/styles/globals.css';
```

## Usage

```tsx
import { Button } from 'nexiumui/components';

export default function MyApp() {
  return (
    <Button variant="primary">
      Click me
    </Button>
  );
}
```
